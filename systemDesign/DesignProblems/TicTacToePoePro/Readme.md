## TIC TAC TOE

Our tic tac toe is the class for our game deriving from game type

we have win strategies like row wise,column wise and diagnoalwise

Besides that we have game state which tell game is in progress,ended or just started
We have match State. telling win draw
we have Manager classes responible for intializing stragegies
